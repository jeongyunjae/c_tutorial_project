### 1번문제
#### 문제내용
다음 소스 코드를 완성하여 5, 4, 1, 250이 각 줄에 출력되게 만드세요.
```
#include <stdio.h>

int main(){
    unsigned char num1 = 1;
    unsigned char num2 = 5;

    printf("%u\n", num1 ___ num2);
    printf("%u\n", num1 ___ num2);
    printf("%u\n", num1 ___ num2);

    num1 = ____ num2;
    printf("%u\n", num1);
}
```
#### 실행결과
**출력**
```
5
4
1
25
```

### 2번문제
#### 문제내용
다음 소스 코드를 완성하여 4가 출력되게 만드세요.
```
#include <stdio.h>

int main(){
    unsigned char num1 = 32;
    num1 = num1 >> 4 << ____;
    printf("%u", num1);
    return 0;
}
```
#### 실행결과
**출력**
```
4
```
